# Tori — Subscriptions & Recurring Billing Engine

> Built for the Nomba × DevCareer Hackathon 2026

Tori is a production-grade managed recurring billing and subscription orchestration
platform built natively on top of Nomba's payment infrastructure.

It is the missing infrastructure layer — what Stripe Billing is to Stripe, Tori is to Nomba.

---

## Quick Start

```bash
cp .env.example .env
# Fill in real values in .env

docker compose up
```

API available at `http://localhost:8080`
Dashboard at `http://localhost:3000`
